# TOOLS — Dostupné nástroje

## Plný přístup
Máš neomezený přístup ke všemu na tomto Mac Mini. Žádné restrikce na adresáře nebo příkazy.

## Povolené nástroje
- **shell** — spouštění příkazů v terminálu Mac Mini (plná práva, přístup kamkoliv)
- **browser** — Brave Browser (výchozí prohlížeč a vyhledávač)
- **files** — čtení a zápis souborů kdekoliv na disku
- **web_search** — vyhledávání na webu
- **web_fetch** — stahování obsahu z URL

## Browser — Brave
- Výchozí prohlížeč: **Brave Browser** (`/Applications/Brave Browser.app`)
- Pro vyhledávání VŽDY používej Brave (Brave Search)
- Brave má built-in ad blocking a tracking protection
- Při web browsing preferuj Brave před jakýmkoliv jiným prohlížečem

## Skills (doinstaluj dle potřeby)
```bash
# Prohlédni dostupné skills
openclaw skills list

# Instalace
openclaw skills install <skill-name>
```
